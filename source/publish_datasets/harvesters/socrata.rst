Socrata harvester
=================

Retrieve datasets from a Socrata portal.

This harvester is not available by default. If you need to use it, please contact OpenDataSoft support team.

Parameters
----------

.. list-table::
   :header-rows: 1

   * * Name
     * Description
     * Example
   * * URL (url)
     * The base URL of the portal
     * https://data.cityofboston.gov
